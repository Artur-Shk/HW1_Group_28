ДЗ СДЕЛАНО НА ОС WINDOWS

1) Посмотреть где я: pwd
2) Создать папку: mkdir
3) Зайти в папку: cd foldername

4) Создать 3 папки: mkdir f_1; 
                                 mkdir f_2;
                                 mkdir f_3;

5) Зайти в любоую папку: cd f_1;
6) Создать 5 файлов (3 txt, 2 json): 
 можно через touch:                         можно через знак >:
 touch a_1.txt;                                    > a_1.txt;
 touch a_2.txt;                                    > a_2.txt;
 touch a_3.txt;                                    > a_3.txt;
touch b_1.json;                                  > b_1.json;
touch b_1.json;                                  > b_1.json;

7) Создать 3 папки: mkdir f_4; 
                                 mkdir f_5;
                                 mkdir f_6;

8. Вывести список содержимого папки: ls (без параметров); ls -la (с параметрами);
9) + Открыть любой txt файл: vim a_1.txt;

10) + написать туда что-нибудь, любой текст: 
на клавиатуре нажать латинскую букву “i”,
ввести текст: qwerty.

11) + сохранить и выйти: 
нажать на клавиатуре клавишу Esc,
поставить знак двоеточия :
ввести сочетание клавиш wq.

12) Выйти из папки на уровень выше: cd ..;
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку: 
mv a_1.txt a_2.txt f_4: 

14) скопировать любые 2 файла, которые вы создали, в любую другую папку: 
cp b_1.json b_2.json f_5;

15) Найти файл по имени: find -name b_2.json;

16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает:
 grep ДЛЯ ./*; (поиск по всем файлам папки)
Ответ: ./a_2.txt:ДЛЯ ЧЕГО МЫ ПОЛЬЗУЕМСЯ GREP-ОМ?
grep ДЛЯ a_2.txt; (поиск по конкретному файлу)
Ответ: ДЛЯ ЧЕГО МЫ ПОЛЬЗУЕМСЯ GREP-ОМ?

17) вывести несколько первых строк из текстового файла:  head -n3 a_2.txt
18) вывести несколько последних строк из текстового файла: tail -n2 a_2.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает: less long.txt; 
20) вывести дату и время: date:
ответ: Sat Mar 12 16:24:49     2022
